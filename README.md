# Bike Sales Dashboard (Excel Portfolio Project)

## Project Summary

This project demonstrates how powerful **Microsoft Excel** can be when used beyond basic spreadsheets.

I built a fully interactive **Bike Sales Dashboard** using raw customer data, transforming it into a structured, insight-driven report. The dashboard focuses on understanding **who buys bikes, what influences their decisions, and how different factors impact sales**.

## Business Problem

Companies need to understand:

* Which customers are more likely to purchase bikes
* How income, age, and commute distance affect buying behavior
* Where to focus marketing and sales efforts

This dashboard answers those questions in a **clear and interactive way**.

## What This Project Demonstrates

### Data Cleaning & Preparation

* Removed inconsistencies and standardized data formats
* Structured raw data for analysis
* Created calculated fields for better segmentation (e.g., Age Brackets)

### Data Transformation

* Organized data into meaningful categories such as:

  * Age groups (Adolescent, Middle Age, Old)
  * Commute distances
* Prepared datasets for efficient pivot analysis

### Analytical Thinking with Excel

Used formulas and logic to derive insights, including:

* Average income per customer group
* Purchase behavior segmentation (Yes/No)
* Customer profiling based on demographics

## Dashboard Features

### Key Visualizations

* **Average Income per Purchase (by Gender)**
  Highlights how income influences purchasing decisions

* **Customer Commute Distance Analysis**
  Shows how distance impacts likelihood of purchase

* **Age Bracket Distribution**
  Identifies which age groups are more likely to buy bikes

### Interactive Elements

* Slicers for:

  * Marital Status
  * Region (Europe, North America, Pacific)
  * Education Level
* Dynamic filtering across all visuals

### Excel Tools Used

* Pivot Tables
* Pivot Charts
* Slicers
* Conditional Formatting
* Excel Formulas for calculated fields

## Key Insights Delivered

From the dashboard:

* Customers with certain income levels show higher purchase likelihood
* Middle-aged customers tend to dominate bike purchases
* Commute distance plays a noticeable role in buying decisions
* Demographics such as education and region influence trends
